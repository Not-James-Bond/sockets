# sockets
Screen Mirror Whiteboard/Blackboard

To Run this
1. Clone this Project
2. Run ``` npm install ```. This will install all the dependencies.
3. Create an env file with required ``` PORT = xxxx ````.
4. Run ``` node start ``` on CMD.
5. Open ``` http://localhost:8001 ``` multiple instances.
6. Doodle on the screen.
